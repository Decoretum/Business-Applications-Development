Web Inventory Order Management Software

Purpose
This software was created to create and manage product transactions for China Global Steel Trading. It could confirm orders and would store and present data in a manner that could be used for the letter of credit and manufacturing order.


Technical Specifications
In order to develop this software, Django (Python), SQL, and Javascript were used.


Some Reminders about Database Logic


Notify Party
Every shipment containing multiple orders would require a Notify Party. The Notify Party would be notified when the shipment arrives at the destination. During the process of confirming an order, there would be a list of choices for available notify parties.

In order to be more efficient, Notify parties will be added in the database of the application in order to save the information regarding these Notify Parties for future use.
